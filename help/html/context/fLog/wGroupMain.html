Log Browser Interface

The Log Browser allows you to view, edit, and search Timekeeper entries.

Note: Moving from row to row (using either the navigation buttons or the search results display automatically saves your changes. If you make an unwanted change, you must either hit Ctrl+Z to undo or press the system close button ("x") in the upper right.

Note: Use YYYY-MM-DD hh:mm:ss date/time formats everywhere in Timekeeper.

Start Time

This control shows the start time of the entry. It can be edited.

Close Start Time Gap

Use the <— button to set the start time value equal to the value of the previous row's end time. This closes any time gaps between the two Timekeeper entries, for example, if you were continuously working and forgot to start the timer again.

End Time

This control shows the end time of the entry. It can be edited. Changing the end time automatically updates the duration of the entry.

Close End Time Gap

Use the —> button to set the end time value equal to the value of the next row's start time. This closes any time gaps between the two Timekeeper entries. If this is the last entry in the file, this sets the end time equal to the current date and time.

Duration

This control shows the duration of the entry in hours, minutes, and seconds (hh:mm:ss). You can edit the duration directly. Doing so automatically updates the end time of the entry.

Pre Log

This displays the annotation (if any) entered when the timer began.

Post Log

This displays the annotation (if any) entered after the timer ended.

Task

This displays the task associated with this entry. To reassign this entry to a new task, select the alternate task from this drop down.

Project

This displays the project associated with this entry. To reassign this entry to a new project, select the alternate project from this drop down.
