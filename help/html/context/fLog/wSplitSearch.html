Log Browser Search

The Search feature allows you to quickly locate Timekeeper entries.

From

This date picker control allows you to specify the start date of your search criteria. It defaults to the date of the first entry in the database.

To

This date picker control allows you to specify the end date of your search criteria. It defaults to the date of the most recent entry in the database.

Text

Enter the search text. Timekeeper uses a very simple "exact match" search. If the word or words you enter appear exactly as entered anywhere in the annotations (between the start and end dates), the matching entry will appear in the results window.
