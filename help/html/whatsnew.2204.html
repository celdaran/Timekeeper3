What's New

Version 2.2

Release Date: May 2, 2011

  • Hidden tasks and projects.
  • Keyboard customizations.
  • Recently used files list.
  • Daily time total added to Log Listing report.
  • Projects added Log Listing report.
  • About box lists stats for the currently opened file.
  • Log Browser now has "close gap" buttons.
  • Stopwatch and Calendar tools cleaned up.
  • Can now edit previously created Journal entries.
  • Help system updated.
  • Grid view remembers last view and automatically reloads it.
  • Customizable title bar text.
  • Can open Timekeeper by double-clicking a Timekeeper data file.
