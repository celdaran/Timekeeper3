Timekeeper Features

Timekeeper allows you to define tasks, record time spent on those tasks, and then let you report on the time spent.

  • Supports an unlimited number of tasks and task folders.
  • Supports projects for two-dimensional timekeeping.
  • Quickly start or stop the timer with Ctrl+T.
  • Easily edit time logged, for making corrections or additions.
  • Time grid, showing total time spent on tasks or projects per day, week, month, or year.
  • A "punch card" report, showing elapsed time between first and last tasks of each day.
  • And more...
