What's New

Version 2.3

Release Date: October 11, 2012

  • 1159: Changed how notifications work (system task bar noficiation bubble).
  • 1177: Enhanced date presets on Grid control (Yesterday, Previous Day, Last Month, Last Year, and a revised This Week).
  • 1179: Fixed modality of report and grid windows.
  • 1190: Added "hours" as a supported time format on Grid control.
  • 1195: Re-wrote grid population from scratch (vast performance improvement, also now sorts alphabetically by default).
  • 1198: The first/last/prev/next buttons use the log entry's timestamp for chronology, not its row identifier.
  • Also: The first/last/prev/next buttons now disable themselves based on current row.
  • 1215: File menu is now disabled while timer is running.
  • 1232: Implemented journal entry count to About box.
  • 1233: Fixed Report ordering.
  • 1235: When a database is closed, all open windows are also closed.
  • Minor bug fixes (See below 1162, 1167, and 1173).
  • Internals: Many functions (primarily database access) now use Technitivity Toolbox.
  • Internals: Target Platform changed from .NET 2.0 to .NET 4.0. Note: Windows 2000 is no longer supported.
